# Android EA Simulator in Python – Learn Trading EA Without a PC

**Description**  
This project is a **simple candlestick simulator** that can run on **Android using Python / Pydroid**.  
The goal is to **learn candlestick charts and EA logic on Android** while also **backing up code to GitHub** for learning purposes and personal documentation.  

---

## Notes
- This is a **personal learning project**, not a professional one  
- All code is created while learning and stored on GitHub for easier development in the future  
- If you have **suggestions or feedback**, please submit them via the GitHub **Issues** feature or contact me through my GitHub profile  

---

## Features

- Simulates a **simple candlestick chart** (Open/High/Low/Close)  
- Can be run directly on **Pydroid / Android**  
- Suitable for learning **candlestick, EA, and simple indicators**

---

## How to Run

- Open **Pydroid 3** on Android  
- Run any Python file in the project  

Result:  
- Candlestick charts appear one by one  
- Green candles = price up, red candles = price down  
- Focuses on learning candlestick formation; no trading logic yet
